Redesigning A Website -

Your Bike Shop RI

I decided to redesign the website for this local Rhode Island bike shop. During the redesign process, I created a style guide for the website as well as low and high fidelity prototypes for various screen sizes to create a more user friendly interface.

I chose this website because it is really outdated and I think a lot of improvements could be made. While it has the important information necessary for a shop, such as locations and hours, it is poorly displayed and doesn't offer a great user experience.

Original Site

Current Issues

Usability/Efficiency

- There is no adaptability between different screen sizes, so it requires panning and zooming to see each part of the page on smaller screens.

- It's hard to find information because the contents are not clearly sectioned.

- The many different typefaces make it confusing and/or overwhelming for the user (i.e. the hours heading is the same size as the body).

Memorability

- There are few to no visual guides that highlight the website's main function/product: a bike shop.

- There is no cohesive design style for the page.

Learnability

- There is no clear hierarchy for the information being displayed which makes it hard for users to find what they are looking for.

- The name of the actual shop is not clear because it's combined with the shop's function in the page header.

Accessibility Issues

I also ran the website through WebAIM WAVE to look at the website's accessibility issues. I found that there was actually no heading stucture in the page at all which was not surprising to me given that each heading has a different typeface (some underlined, some bolded, some larger, etc.). I also found that most of the text was too small and that some of the functions were device dependent which is not very user friendly.

Low-fi Prototypes

I took into account all of the issues with the current interface and created low fidelity mock ups for a redesigned version of the website. I created these prototypes for 3 different screen sizes: a laptop, a tablet, and a phone.
